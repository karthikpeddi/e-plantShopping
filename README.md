# e-plantShopping

## Paradise Nursery Shopping Application

This repository (**e-plantShopping**) contains the final project for building a simple e-commerce shopping cart experience for an online plant shop called **Paradise Nursery**.

### Features
- Landing page with a **Get Started** button
- Product listing page with plant categories and **Add to Cart** functionality
- Shopping cart page with:
  - Increase/decrease quantity
  - Remove items
  - Dynamic cart total
  - **Continue Shopping** and **Checkout (Coming Soon)**
- Global state management using **Redux Toolkit**

### Run locally
```bash
npm install
npm run dev
```
